การใช้ Lightroom เบื้องต้น

1.เปิดโปรแกรม Adobe Photoshop Lightroom

 

2.คลิก Import เพื่อนำรูปภาพเข้าโปรแกรม

 

3.เลือกรูปภาพ หรือโฟลเดอร์ของรูปภาพ สามารถ Import ทั้งโฟลเดอร์ได้ จากนั้นคลิกที่ปุ่ม Import

 

4.เลือก Develop

 

5.คลิกสามเหลี่ยม Show Hightlight Clipping และ Show Shadow Clipping เพื่อแสดงส่วนที่สว่างเกินไปจนมองไม่เห็นรายละเอียดในส่วนนั้น

 

6.ปรับค่า Hightlight เพื่อให้เห็นรายละเอียดในส่วนที่สว่างเกินไป

 

7.ปรับค่าความสว่างและสีของภาพตามความเหมาะสม

 

8.หากต้องการ Copy การตั้งค่าไปใช้กับภาพอื่นๆ ให้คลิกที่เมนู Settings เลือก Copy Settings...

 

9.คลิกภาพถัดมาที่ต้องการตั้งค่าเหมือนภาพแรก

 

10.คลิกที่เมนู Settings เลือก Paste Settings from Prevoius เพื่อวางการตั้งค่า

 

11.นำไฟล์ออกไปใช้งานโดยเข้าเมนู File เลือก Export

 

12.เลือกโฟลเดอร์ที่ต้องการจัดเก็บไฟล์และสามารถเปลี่ยนชื่อไฟล์ได้

 

13.สามารถใส่ลายน้ำลงในภาพได้ในส่วน Watermarking คลิก Edit Watermarks
